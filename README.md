# Where Does California Turn Green First?

An interactive D3.js visualization exploring seasonal vegetation patterns across California using MODIS NDVI and EVI data.

## 📊 Overview
This project investigates which regions of California “turn green” first each spring by visualizing satellite-derived vegetation indices over time.

## 🗺️ Features
- Interactive choropleth map of California
- Month slider with animation
- NDVI / EVI toggle
- Hover tooltips
- Click-to-county time series

## 🧠 Key Insight
Northern coastal regions (e.g., Humboldt, Mendocino) consistently green up earlier than inland and desert regions, which experience delayed vegetation growth.

## 📁 Data
- MODIS MOD13A3 (Vegetation Indices)
- Monthly NDVI & EVI at 1km resolution
- Aggregated to county level
